# Krzycz Trybson

## How to prepare for development
1. Use `Pyhton 3.11.X`
2. Clone the repository
3. Create a Poetry virtual environment
4. Install dependencies with `poetry install`
5. Install torch with CUDA support `pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu129`
6. (Optional, but recommended) Use `pre-commit install` to set up pre-commit hooks
7. Run the application with `poetry run python main.py`

Please consult the [CONTRIBUTING.md](CONTRIBUTING.md) file for more details on how to contribute to the project.
